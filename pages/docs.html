Paso 1: Crear tu Wallet

Una wallet es tu identidad en la blockchain. Te permite recibir, enviar y almacenar RSC tokens.

JavaScript 

// Crear wallet usando la API
const response = await fetch('/api/wallet/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  }
});

const wallet = await response.json();
console.log('Wallet creada:', wallet.address);
